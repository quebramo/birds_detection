# YOLOv8 Object Detection and Tracking

## Описание проекта

Этот проект использует модель YOLOv8 для детекции объектов и их трекинга на видео. Он включает:

- Обучение модели YOLOv8 на собственном датасете.
- Трекинг объектов на видео с помощью модели YOLOv8 и алгоритма BotSort.
- Простое использование с минимальными настройками для воспроизведения или работы с собственными данными.

## Стек технологий

- **YOLOv8** — для детекции объектов.
- **OpenCV** — для работы с видео.
- **PyTorch** — для обучения и использования модели.
- **BotSort** — для отслеживания объектов.

## Структура проекта

Проект состоит из нескольких основных компонентов:

- **train.py** — скрипт для обучения модели на вашем датасете.
- **track.py** — скрипт для трекинга объектов на видео с использованием обученной модели.
- **data.yaml** — конфигурационный файл с настройками датасета для обучения.
- **requirements.txt** — файл с зависимостями проекта.
